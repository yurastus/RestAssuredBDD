############### DESCRIPTION #################
Project includes example of automated tests for fake rest service
using cucumber and rest-assured dependencies


################## USAGE ####################
1) Run tests from command line (PARALLEL MODE)
    use cli command: 'mvn clean verify'
    to see reports open: 'output/cucumber-parallel/<FEATURE-INDEX>/index.htm'
    (parallel mode can be modified in pom.xml)

2) Run tests using test runner class
    run tests using 'APITestsRunner.java'
    to see reports open: output/index.html
